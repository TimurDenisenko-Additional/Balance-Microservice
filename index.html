API Documentation

Base URL

http://ls2.denisenko.com

Endpoints

GET /users

Retrieve a list of all users.

Response:

[
    {
        "id": 1,
        "username": "JohnDoe",
        "email": "john.doe@example.com",
        "balance": 100.00,
        "currency": "USD"
    },
    ...
]

GET /users/:id

Retrieve details of a specific user by ID.

Response:

{
    "id": 1,
    "username": "JohnDoe",
    "email": "john.doe@example.com",
    "balance": 100.00,
    "currency": "USD",
    "transaction_history": [
        {
            "type": "Deposit",
            "amount": 50,
            "date": "2024-01-01T12:00:00Z"
        }
    ]
}

POST /users

Create a new user.

Request Body:

{
    "username": "JohnDoe",
    "email": "john.doe@example.com",
    "balance": 100.00,
    "currency": "USD"
}

Response:

{
    "id": 1,
    "username": "JohnDoe",
    "email": "john.doe@example.com",
    "balance": 100.00,
    "currency": "USD"
}

POST /users/:id/deposit

Add funds to a user account.

Request Body:

{
    "amount": 50.00
}

Response:

{
    "message": "Deposit successful",
    "new_balance": 150.00
}

POST /users/:id/withdraw

Withdraw funds from a user account.

Request Body:

{
    "amount": 20.00
}

Response:

{
    "message": "Withdrawal successful",
    "new_balance": 80.00
}

POST /users/:id/transfer

Transfer funds from one user to another.

Request Body:

{
    "recipient_id": 2,
    "amount": 30.00
}

Response:

{
    "message": "Transfer successful",
    "sender_new_balance": 70.00,
    "recipient_new_balance": 130.00
}

GET /users/:id/transactions

Retrieve the transaction history of a user.

Response:

[
    {
        "type": "Deposit",
        "amount": 50.00,
        "date": "2024-01-01T12:00:00Z"
    },
    {
        "type": "Withdrawal",
        "amount": 20.00,
        "date": "2024-01-02T15:00:00Z"
    }
]

DELETE /users/:id

Delete a user by their ID.

Response:

{
    "message": "User deleted successfully."
}
