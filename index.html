API Documentation

Base URL

http://ls2.denisenko.com/api

Endpoints

GET /users

Retrieve a list of all users.

Response:

[
    {
        "id": 1,
        "username": "JohnDoe",
        "email": "john.doe@example.com",
        "balance": 100.00,
        "currency": "USD"
    },
    ...
]

POST /users

Create a new user.

Request Body:

{
    "username": "JohnDoe",
    "email": "john.doe@example.com",
    "balance": 100.00,
    "currency": "USD"
}

Response:

{
    "id": 1,
    "username": "JohnDoe",
    "email": "john.doe@example.com",
    "balance": 100.00,
    "currency": "USD"
}

PUT /users/:id

Update an existing user.

Request Body:

{
    "username": "UpdatedName",
    "email": "updated.email@example.com",
    "balance": 150.00,
    "currency": "EUR"
}

Response:

{
    "id": 1,
    "username": "UpdatedName",
    "email": "updated.email@example.com",
    "balance": 150.00,
    "currency": "EUR"
}

DELETE /users/:id

Delete a user by their ID.

Response:

{
    "message": "User deleted successfully."
}
